Centos下安装docker

系统要求：64为操作系统，内核版本至少为3.10\
docker目前支持Centos6.5及以后的版本

yum安装

执行以下添加docker的yum源

对于centos7系统，Centos-Extras源中已经内置了docker，如果已经配置了Centos-Extras源,可以直接通过命令yum install -y docker安装
